# Google Stitch Design Prompt

Copy-paste this into Google Stitch (stitch.withgoogle.com) to generate the visual design.

---

## Prompt for Home Screen

```
Design a mobile app home screen (375x812) for a Hebrew RTL quest/RPG game called "מסע הגיבורים שלי" (My Hero's Journey). Dark theme with gradient background (#0f0e17 to #1a1a2e).

At the top: large gradient title text (orange #ff8906 to pink #e53170), subtitle "לקראת בר מצווה", and an XP progress bar showing "100 / 1000 XP".

Below: a vertical quest map showing 10 quest nodes connected by lines. Each node is a card with an emoji icon on the right, quest name and subtitle in the center. The first quest is completed (green border, checkmark), the second is active (orange pulsing border), and quests 3-10 are locked (dimmed, lock icon).

Quest names in order: שורשים 🌳, משחק המשפחה 🎮, הסיפור שלפניי 📖, הלוחם הקטן ⚔️, המוח שלי עובד אחרת 🧠, תאום אבל אני 👫, הדרך שעשיתי 🏆, הסופרפאוורס שלי ⚡, האנשים שלי 🤝, מי אני עכשיו 🌟

Style: gaming RPG interface, clean modern design, rounded corners, subtle glow effects. Hebrew text right-to-left.
```

## Prompt for Quest Screen

```
Design a mobile quest detail screen (375x812) for a Hebrew RTL identity quest app. Dark theme (#0f0e17).

Top sticky header: back arrow, quest emoji badge (green circle with 🌳), quest title "שורשים".

Content area with cards:
1. Introduction card with gradient border (orange to pink) explaining the quest
2. Identity message card: "אני חלק מסיפור גדול"
3. Family member grid (2x3) - each cell has a circular avatar with first letter, name below, and a text input for one word
4. Reflection text area: "מה הדבר המשותף לכל המשפחה שלך?"

Bottom: sticky gradient complete button "סיימתי את המשימה! ✓" in orange-to-pink gradient.

Style: RPG game UI, dark cards on darker background, rounded corners (16px), Hebrew RTL, Heebo font.
```

## Prompt for XP Popup

```
Design a full-screen overlay/modal for a quest completion celebration. Dark semi-transparent background.

Center: large emoji (🌳), "משימה הושלמה!" heading, "+100 XP" in large gradient text (orange to pink), italic message "אני חלק מסיפור גדול", and "המשימה הבאה נפתחה! 🔓" in green.

Style: celebration, achievement unlock feel, clean animation-ready design. Hebrew RTL.
```

## Prompt for Hero Book

```
Design a mobile "Hero Book" view (375x812) - a digital book showing completed quest results. Dark theme.

Header: back button, title "ספר הגיבור שלי".

Content: scrollable pages/cards, each representing a quest chapter:
- Title page: "ספר הגיבור שלי" + "גיא" + "בר מצווה תשפ״ו"
- Chapter cards with colored header (quest color), emoji, chapter title, italic identity message, and text responses below

Footer: two outline buttons - "ייצוא לPDF 📄" and "צפייה במצגת 🎬"

Style: book/journal aesthetic within dark RPG theme, elegant, printable. Hebrew RTL.
```
